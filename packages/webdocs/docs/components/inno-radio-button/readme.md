# inno-radiobutton

Radio button for the Innomics applications.

## Classes

| Name               | Description            |
| ------------------ | ---------------------- |
| `inno-radiobutton` | `design wrapper class` |
| `inno-dark`        | `dark mode class`      |

## How to use

Wrap the given input
and add the necessary classes to the wrapper.

### Light mode example.

```html
<div class="inno-radiobutton">
  <input type="radio" id="id1" />
  <label for="id1">label</label>
</div>
```

### Dark mode example.

```html
<div class="inno-radiobutton inno-dark">
  <input type="radio" id="id1" />
  <label for="id1">label</label>
</div>
```
